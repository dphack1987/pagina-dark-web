Descargar Proyecto Completo

Instrucciones de Instalación:

  1. Descarga el archivo ZIP
  2. Extrae el contenido en tu computadora
  3. Instala Docker Desktop para Windows
  4. Ejecuta setup-onion.bat como administrador
  5. Espera a que se genere tu dirección .onion

⚠️ Asegúrate de tener Docker Desktop instalado y en ejecución antes de iniciar la instalación
